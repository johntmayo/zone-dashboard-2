1. What Discord Is

Discord is a group-chat platform (similar to Slack or WhatsApp) built for larger communities. You can use it in your web browser or as an app on your phone or tablet.

Everything is organized into:

  • Servers â€“ big containers for communities (you'll mainly use the Eaton Fire Survivors Network server).
  • Categories â€“ folders that group related topics.
  • Channels â€“ individual chat rooms (insurance, remediation, general, etc.).

Neighborhood Captains also get access to a private Neighborhood Captain category once we assign your role.

2. Basic Layout

  • Left column: Your Direct Messages (DMs) and the list of servers you're in.
  • Middle column: Categories and channels for the server you're viewing.
  • Main panel: The chat, emoji reactions, and the message box.

In the message box you can:

  • Use @name to tag someone and send them a notification.
  • Use #channel to reference another channel.

3. Customizing Which Channels You See

To reduce clutter and focus on what matters to you:

  1. Click the server name at the top of the channel list.
  2. Select "Channels & Roles" and then "Browse Channels".
  3. For each category, decide if you want to:
    • Leave "Follow Category" on to see all channels in that category, or
    • Turn "Follow Category" off and individually toggle the channels you care about.
  4. Changes save automaticallyâ€”there's no save button.
  5. Go back to the server view to see your simplified channel list.

4. Editing Your Profile

  1. Click your profile area in the bottom left of Discord.
  2. Select "Edit Profile".
  3. Update your:
    • Display name
    • Pronouns (optional)
    • Avatar (profile picture)
    • About Me (short bio)
  4. Click "Save Changes" when you're done.

Discord Nitro (paid) unlocks cosmetic extras only. All the core features you need are included in the free version.

Tip: Neighborhood Captains often add their Zone number to their display name.

5. Display Name vs Username vs Server Nickname

  • Username: Your global Discord identity (often includes numbers). This rarely changes.
  • Display Name: The name most people see when you chat on servers.
  • Server Nickname: An optional name that only applies inside a specific server.

If you're only using Discord for the Eaton Fire Survivors Network and Altagether, you can ignore server nicknamesâ€”just set a clear display name and you're good.

6. Adding Friends & Getting NC Access

Important: After joining the Discord server, message @altagether_org with your name so we can add you to the private NC channel.

You'll also need to add the Altagether account as a friend so we can assign your Neighborhood Captain role.

Method 1: From a Post

  1. Click on the person's name anywhere you see it.
  2. Click "Add Friend".

Method 2: By Searching

  1. Go to Direct Messages in the left column.
  2. Click "Friends".
  3. Click "Add Friend".
  4. Type their exact username and send the request.
  5. Check the Pending tab to see requests you've sent.

Once we accept your friend request and you've messaged us, we'll assign your Neighborhood Captain role so you can see the private NC channels.

7. Notifications

You can control how often Discord pings you:

  • Right-click a channel and choose "Mute" to silence that channel.
  • Mute an entire category if it's not relevant to you.
  • You'll still see unread indicators, but you won't get sound or push notifications.

8. Light Etiquette

  • Try to post questions in the channel that best matches your topic.
  • Use threads when they're available to keep conversations tidy.
  • Be kind and patientâ€”everyone's navigating a lot.
  • No question is too small; this space exists to support you.
